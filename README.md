# Mind Hand

スマホを数字カードとして使う、通信なしの協力ゲーム補助アプリです。

## 使い方

1. iPad miniで「親機」を開き、プレイヤー数と手札枚数を選びます。
   デザインは「プレーン」か「マインド」を選べます。
2. 親機の「公開」で合言葉、QR、確認コードを出します。
   ローカルPCから試す場合は「QRの接続先」を開き、スマホから開けるURLになっているか確認します。
3. iPhone側はQRを読むか、「子機」でプレイヤー番号、人数、枚数、合言葉を合わせます。
4. 子機の確認コードが親機と同じなら開始します。
5. 子機は自分の最小カードをタップして公開します。
6. 親機は場に出たタイミングで画面をタップし、正しい次カードを公開します。
7. 数字が違ったら親機の「失敗」で振り返り画面に進みます。
8. 最後まで出せた場合も振り返り画面に進みます。
9. 振り返り後に「次のゲーム」を押し、新しい合言葉を合わせます。

## 開発

```bash
npm install
npm run dev
```

## 配布

```bash
npm run build
```

生成された `dist` フォルダを GitHub Pages、Cloudflare Pages、Netlify などの無料静的ホスティングに置けば使えます。PWA対応なので、一度開いた端末ではオフラインでも読み込めるようになります。
